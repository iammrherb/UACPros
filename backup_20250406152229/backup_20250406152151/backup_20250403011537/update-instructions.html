Testing Your Implementation

After completing the installation, verify that the following features are working correctly:

  1. Accordion Functionality - Ensure all accordion sections open and close properly with the correct icons
  2. Tab Navigation - Verify that all tabs, including the new Environment Profile and Portnox Cloud tabs, navigate correctly
  3. Vendor-Specific Options - Check that the appropriate vendor-specific options appear when selecting different vendors and platforms
  4. Environment Profiling - Test the environment discovery section to ensure it correctly captures and displays your infrastructure
  5. Portnox Integration - Verify that all Portnox Cloud features are accessible and responsive

Note: The Portnox Cloud integration requires valid API credentials. If you don't have access to Portnox Cloud, the system will still function in demonstration mode with simulated responses.

Success Criteria

Your implementation is successful when:

  • All accordion sections expand and collapse properly
  • Vendor-specific options are displayed correctly based on selections
  • Environment discovery captures and displays infrastructure details
  • POC templates can be generated based on environment profiles
  • Configurations are correctly generated for all supported vendor platforms

Further Customization

The system is designed to be easily extended. To add support for additional vendors:

  1. Create vendor-specific options in the HTML
  2. Add the vendor to the updatePlatformOptions() function in js/vendors.js
  3. Create configuration templates in the assets/vendors/ directory
